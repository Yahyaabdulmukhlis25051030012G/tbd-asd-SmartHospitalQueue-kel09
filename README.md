## 📌 Deskripsi Sistem

Sistem antrian rumah sakit cerdas yang mensimulasikan:
- **Priority Queue** — antrian 5 poli dengan prioritas KRITIS, PRIORITAS, REGULER
- **Stack** — riwayat tindakan dokter per poli dengan fitur undo
- **BST** — pencarian rekam medis cepat O(log n)
- **Sorting** — laporan harian dengan Insertion Sort & Selection Sort
